A página em si é um implemento simples de uma página de avaliação de cursos utilizando o JavaScript como um simulador de back-end.

Para a simulação foram utilizadas as seguintes estruturas: forEach nos cursos definidos no próprio arquivo js e for para identificar os botões no nosso html e substituí-los por botões pintados

Por fim é válido destacar que utilizamos o localStorage para armazenar a avaliação de cada curso fazendo a substituição de estrelas ao recarregar a tela com a função implementada no window.onload
